## System tools available

You have access to these CLI tools via the shell tool:

- `git` — version control (status, diff, add, commit, push, log)
- `curl` + `jq` — HTTP requests and JSON processing
- `find`, `grep`, `sed`, `awk` — file search and text processing
- `tar`, `gzip` — archiving
- `ssh`, `scp` — remote access
